# Системные требования
### JAVA - 17
### СИСТЕМА СБОРКИ - MAVEN версия 3.9.11
### СТОРОННИЕ БИБИЛИОТЕКИ - нет

## ИНСТРУКЦИЯ ПО СБОРКЕ
1) Зайти в рабочую директорию с исходным кодом, где находится папка - src и файл pom.xml
2) Вызвать maven c параметрами clean install - обычно: mvn clean install
3) Появится папка target, в ней будет TEST_TASK.jar

## ИНСТРУКИЦИЯ ПО ЗАПУСКУ
1) Зайти в папку target и найти в ней TEST_TASK.jar
2) Прописать:  java -jar TEST_TASK.jar (далее здесь можно прописать параметры например полные имена входных файлов или флаги)

## Параметры запуска
1) -s - вывод краткой статистики
2) -f - вывод полной статистики(при указании одновременно и полной и краткой статистики выводится полная)
3) C:\input\input.txt - полное имя входного файла. Если программа не может открыть файл, он просто пропускается. Предупреждение выводится в командную строку
4) -o C:\input - путь для выходных файлов. Если пусть не указан - используется текущая папка. Если не удалось создать или открыть файл по пути - программа падает.
5) -p prefix_ - префикс для выходных файлов.
6) -a - Режим добавления в существующие файлы. Если файла нет - выводится предупреждение.
